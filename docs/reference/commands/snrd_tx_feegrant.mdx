# snrd tx feegrant

Grant and revoke fee allowance for a grantee by a granter

## Usage

```shell
snrd tx feegrant [flags]
```

```shell
snrd tx feegrant [command]
```

## Available Commands

| Command | Description |
|---------|-------------|
| `grant` | Grant Fee allowance to an address |
| `prune` | Prune expired allowances |
| `revoke` | Revoke a fee grant |
